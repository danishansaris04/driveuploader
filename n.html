Save fail ho raha hai? Ye 2 cheezein check karein:
1. Realtime Database → Rules (Firebase Console)
{
  "rules": {
    "posters": {
      ".read": true,
      ".write": "auth != null"
    }
  }
}
2. Storage → Rules (agar image upload use kar rahe ho)
rules_version = '2';
service firebase.storage {
  match /b/{bucket}/o {
    match /posters/{file} {
      allow read;
      allow write: if request.auth != null;
    }
  }
}
